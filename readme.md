# KyanitePackTool

## 📋 项目简介
**KyanitePackTool** 是一款功能强大的 Minecraft 材质包处理工具，提供直观的图形用户界面，支持材质包转换、覆盖包制作等多种功能。本工具由 **Hurricane Centre** 开发维护。

## ✨ 主要功能

### 材质包转换
- 支持 Minecraft 版本 **1.6.0 - 1.21.11**
- 跨版本材质包格式转换
- 智能资源适配和优化

### 覆盖包制作
- 创建自定义覆盖包

### 设置功能
- 批量处理选项

## 🚀 安装与使用
**支持的平台：**
- Windows 10 1803+ & 11
- Linux (In Plan)

## 📖 使用指南

### 1. 材质包转换
1. 打开程序并选择 **"材质包转换"** 功能
2. 选择源材质包文件或文件夹
3. 选择目标 Minecraft 版本
4. 点击 **"开始转换"** 按钮

### 2. 覆盖包制作
1. 选择 **"覆盖包制作"** 功能
2. 修改需要的功能选项
3. 生成并保存覆盖包

### 3. 设置配置
- 在设置界面调整有关程序内容的配置
